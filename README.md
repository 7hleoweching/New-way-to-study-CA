# 💻 BCA Computer Architecture Masterclass (Modules 1 & 2)
### 🎓 Batch 2025 - 2029 | Mid-Term Strategic Hub

Welcome to the ultimate digital study hub for the BCA 2nd Semester Computer Architecture (CA) Mid-Terms. This project was built to simplify complex hardware concepts into digestible, visual, and high-impact learning modules.

## 🚀 Project Goals
* **Accessibility:** Making CA easy to understand for everyone in the batch.
* **Scientific Accuracy:** Covering deep-dive topics like IEEE 754, 2's Complement Logic, and Flip-Flop Toggling.
* **Visual Learning:** Integrated diagrams for K-Maps, Logic Gates, and Sequential Circuits.

## 📚 What's Inside?
- **Module 1:** Binary Arithmetic, Number Systems, and Floating Point Representation.
- **Module 2:** Combinational vs. Sequential Circuits, Universal Gates, and Memory Elements.
- **Interactive Quiz:** A rapid-fire self-assessment tool for last-minute revision.

## 🛠️ Built With
* HTML5
* CSS3 (Glassmorphism & Responsive Design)
* Educational API Hotlinking for Diagrams
