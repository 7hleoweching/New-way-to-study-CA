Module 1: Number Systems & Arithmetic

Q: Why do we use Hexadecimal in Computers?

Binary (01010111) is hard for humans. Hexadecimal is a shorthand. Every 1 Hex digit perfectly replaces 4 Binary bits. It saves space and prevents errors.

⚠️ Common Mistake: Forgetting that A=10, B=11, C=12, D=13, E=14, F=15.
Q: IEEE 754 Floating Point (32-bit)

The standard way to store decimals. Remember the Bias of 127!

Module 2: Logic Gates & Design

Q: Universal Gates (NAND/NOR)

You can create AND, OR, and NOT gates using only NAND gates. This is why manufacturers love them—it makes production cheaper.

Q: K-Map Optimization

The goal is to simplify Boolean Algebra to the smallest possible circuit.

Q: JK Flip-Flop & Race Around Condition

A sequential circuit that remembers 1 bit. When J and K are both 1, it toggles (flips). To stop "Race Around" (unstable flipping), we use Master-Slave Flip-Flops.

🎯 Rapid Exam Quiz

1. How many entries are in a 4-variable K-Map?

Answer: 16 (24).

2. Which code is used in K-Map labeling?

Answer: Gray Code (only 1 bit changes).

3. What happens in an SR Flip-Flop if S=1 and R=1?

Answer: Invalid/Forbidden state!
